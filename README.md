# ☕ Java Spring Reference

Repositório com projetos práticos desenvolvidos durante meus estudos na trilha **Java Spring Reference**, com foco em recursos importantes do ecossistema Spring e na aplicação prática de conceitos de back-end com Java.

## 📚 Sobre o repositório

Este repositório reúne projetos de estudo voltados para diferentes abordagens dentro do ecossistema Spring, incluindo:

- modelagem de domínio
- bancos NoSQL
- programação reativa
- processamento em lote com Spring Batch

A proposta aqui é registrar minha evolução prática com Java e Spring por meio de exemplos, workshops e pequenas aplicações organizadas por tema.

## 🗂️ Estrutura do repositório

Atualmente, este repositório contém os seguintes projetos:

| Projeto | Tema principal |
|---|---|
| `hello-world-spring-batch` | Introdução ao Spring Batch |
| `modelo-dominio-complexo` | Modelagem de domínio complexa com ORM |
| `workshop-webflux-mongo` | Programação reativa com Spring WebFlux e MongoDB |
| `workshopcassandra` | Persistência com Cassandra |
| `workshopmongo` | Persistência com MongoDB |

## 🚀 Projetos em destaque

### `modelo-dominio-complexo`

Projeto com foco em **modelagem de domínio complexa**, explorando relacionamentos entre entidades e organização de um modelo mais próximo de cenários reais de negócio.

#### Modelo conceitual

Esse projeto é especialmente relevante por demonstrar a construção de um domínio com múltiplas entidades e associações, reforçando conceitos importantes de:

- entidades e relacionamentos
- ORM com JPA
- organização do domínio
- estruturação de regras de negócio a partir do modelo conceitual

### `workshopmongo`

Projeto voltado para estudos com **MongoDB**, trabalhando conceitos de persistência NoSQL e consultas em documentos.

### `workshopcassandra`

Projeto de prática com **Cassandra**, explorando uma abordagem NoSQL diferente da relacional, com foco em modelagem orientada a consultas.

### `workshop-webflux-mongo`

Projeto que combina **Spring WebFlux** com **MongoDB**, explorando conceitos de programação reativa e construção de APIs reativas.

### `hello-world-spring-batch`

Projeto introdutório para estudo de **Spring Batch**, com foco na compreensão inicial da estrutura e funcionamento de processamento em lote.

## 🛠️ Tecnologias trabalhadas

Entre as principais tecnologias e conceitos utilizados nos projetos deste repositório, destacam-se:

- Java
- Spring Boot

- Spring WebFlux
- Spring Batch
- MongoDB
- Cassandra

## 🎯 Objetivo

Este repositório tem como objetivo consolidar conhecimentos práticos em Java e Spring, por meio da implementação de projetos que abordam diferentes estilos de desenvolvimento e persistência de dados.

Além de servir como registro da minha evolução nos estudos, ele também funciona como portfólio técnico dos temas que já pratiquei diretamente.
